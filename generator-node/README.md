# generator-node

## Installation

First, install [Yeoman](http://yeoman.io) and generator-node using [npm](https://www.npmjs.com/) (we assume you have pre-installed [node.js](https://nodejs.org/)).

```bash
npm install -g yo
npm link
```

Then generate your new project:

```bash
yo node
```
